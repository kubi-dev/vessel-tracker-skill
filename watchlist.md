## Data Sources (free, no login required)

1. **ShipInfo.net** — `https://shipinfo.net/vessels_map_imo_[IMO]` — Best for track points with timestamps
2. **Maritime Optima** — `https://maritimeoptima.com/public/vessels/pages/imo:[IMO]/mmsi:[MMSI]/[NAME].html` — Port calls, destination, ETA
3. **VesselTracker** — `https://www.vesseltracker.com/en/Ships/[Name]-[IMO].html` — Current status, nav status
4. **MyShipTracking** — `https://myshiptracking.com/vessels/[name]-mmsi-[MMSI]-imo-[IMO]` — Coordinates, movement events
5. **MagicPort** — `https://magicport.ai/vessels/tanker/[name]-mmsi-[MMSI]` — Owner, manager, sanctions
